# 🚀 AIH Portfolio – Futuristic Developer Portfolio

A modern and futuristic **personal portfolio website** built using **HTML, CSS, and JavaScript**.  
It showcases projects, skills, professional profile, and contact information with **advanced animations, interactive UI, and responsive design**.

The project is **lightweight, fast, and framework-free**, making it easy to customize and deploy.

# 📌 Features

## 👨‍💻 Hero Section
- Dynamic **typing animation** for role titles
- Animated hero cards and metrics
- Smooth hover interactions and call-to-action buttons

---

## 🌌 Skills Universe
- **Rotating circular orbit visualization**
- Skill nodes displayed in a **solar system style**
- Tooltip on hover showing skill category
- Skills loaded from a **centralized configuration file**

---

## 📂 Projects Section
- Responsive **project showcase cards**
- **Modal popup** displaying:
  - Project image
  - Description
  - Technologies used
  - GitHub / Live links
- Automatic project image mapping (supports up to **20 images**)

---

## 🎨 Theme System
- **Light Mode / Dark Mode Toggle**
- Theme preference saved using **localStorage**
- Theme-aware UI including:
  - Backgrounds
  - Cards
  - Text
  - Buttons

---

## ✨ Scroll Animations
- Section reveal animations on scroll
- Direction-based animation variants
- Staggered animation effects
- **Reduced motion accessibility support**

---

## 📱 Responsive Design

Fully optimized for:

- Desktop 💻
- Tablet 📟
- Mobile 📱

Includes:
- Mobile navigation menu
- Adaptive layouts
- Responsive orbit animations
