# 🚀 AIH Portfolio — Cosmic Sync Edition

A high-performance, cinematic **Personal Portfolio** built for the future. This project features a bespoke "Cosmic Sync" entry experience, advanced SVG animations, and a sophisticated minimalist aesthetic designed to leave a lasting professional impression.
---

## ✨ Signature Features

### 🎬 Cinematic Entry Gate
- **High-Fidelity "Cosmic Sync" Loader**: A premium intro sequence featuring a bespoke animated monogram that "draws" itself in real-time.
- **Dynamic Meteor System**: A JavaScript-driven particle engine generating realistic falling stars with randomized trajectories, high-velocity flicker, and aerodynamic scaling.
- **Tactical Holographic UI**: Symmetrical futuristic side-panels featuring dynamic data-streams, precision corner brackets, and a horizontal "scanning" beam.
- **Atmospheric Background**: A vertical starfield drift layered with a pulsing hexagonal tactical grid and dual-toned nebula glows.
- **Gatekeeper Logic**: A robust redirection system that verifies session tokens, ensuring a seamless transition from the brand intro to the core portfolio.

### ✒️ Custom Monogram Branding
- **Bespoke SVG Monogram**: A custom-designed "AIH" logo leveraging advanced CSS `stroke-dasharray` and `dashoffset` for an elegant, hand-drawn reveal.
- **Minimalist Grayscale Palette**: A curated aesthetic using **Brushed Silver (#888)**, **Pure White (#FFF)**, and **Deep Void (#050508)**.
- **Interactive Nav Mark**: A persistent brand signature in the navigation bar with a spring-loaded `cubic-bezier` hover animation and synchronized neon glows.

### 🌌 Skills Universe
- **3D Orbital Visualization**: A perspective-shifted solar system layout where skills orbit a central node.
- **Responsive Physics**: Orbit radii and rotation speeds dynamically adjust based on screen size to maintain visual harmony on mobile and desktop.
- **Centralized Skill Manager**: Scalable JSON-like configuration via `skillManager.js` for instant content updates.

### 📂 Pro-Level Showcase
- **Glassmorphic Project Cards**: Utilizing `backdrop-filter: blur()` and linear gradients to achieve a high-end frosted glass effect.
- **Holographic Modals**: Immersive project deep-dives with automatic image mapping, categorized tech-tags, and action-oriented CTAs.
- **Custom Typing Engine**: A high-performance typing animation for professional roles, optimized for zero layout shift.
